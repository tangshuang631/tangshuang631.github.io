Ayes 架构图

面向 Agent 的本地桌面监控、视觉理解与记忆检索架构

Agent 优先交互

  • • 默认先使用轻量 query，不直接读取原始证据
  • • 菜单栏只承担辅助控制与状态查看
  • • 任务配置可通过对话持续调整

采集处理链路

  • • 采样支持屏幕、进程、窗口和 ROI
  • • OCR 与可选 VLM 统一沉淀为事实记忆
  • • 去重与压缩降低 token 与磁盘成本

本地持久化

  • • 运行数据存放在已安装 skill 目录下
  • • 每个任务独立保存配置、记忆、日志和截图
  • • SQLite / FTS 索引支持低 token 检索
