项目概览

项目定位

安装到 Codex / OpenClaw 后可直接使用的本地视觉监控 skills

核心目标

让监控结果更像可对话的事实记忆 而不是一堆原始截图和日志

技术亮点

统一事实层

通过 MemoryFact 把 event、OCR、VLM 摘要统一成可检索事实 再写入短期记忆和 FTS 索引

低 token 查询

默认 query 先读轻量索引和压缩记忆 避免把原始 JSONL、截图块和日志直接塞给模型

任务分层

支持主任务、ROI 子任务、专属配置和独立目录树 便于长期监控和多任务并行

本地增强

可接入 Ollama 视觉模型 做 OCR 抗噪和主内容理解 同时保留离线和低资源路径

设计与架构

设计风格

延续 OpenCow 的克制静态作品集语言 以分割线和模块序列组织信息 不做花哨卡片堆叠

架构思路

技能入口、运行态、发行态和安装态分离 让项目既能被安装使用 也能持续迭代和发布
