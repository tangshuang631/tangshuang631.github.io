基于 OpenClaw 的中度改造：桌面工作台、Ollama、本地任务执行、权限安全、审计回退与扩展能力统一收口。

桌面产品层

  • • React 工作台承载会话、任务、来源和状态。
  • • Tauri/Rust 负责原生文件、Ollama、附件和系统能力。
  • • 中文 UI 面向本地工作流，降低 OpenClaw 原始使用成本。

OpenClaw 适配层

  • • `openclaw-adapter` 对接模型、工具、任务和扩展能力。
  • • RAG、Skills、NPC、MCP 统一进入能力编排链路。
  • • 上游源码保持隔离，后续深改先在适配层验证。

安全与恢复

  • • 本地执行遵循预览、确认、执行、验证、审计、回退。
  • • 高风险 Shell 和文件动作经过权限与安全引擎判断。
  • • 会话、审计、快照和知识索引默认保留在本地。
